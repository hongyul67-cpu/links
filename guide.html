📥 결과수집 설정 가이드

학생들이 학습도구를 풀면, 결과가 내 구글 시트에 자동으로 쌓이게 만듭니다.

💻 1~4단계는 컴퓨터로 딱 한 번만 하면 돼요(5분). 그 뒤 학생 링크 보내기·결과 보기는 폰으로 됩니다.
※ "AppSheet" 앱은 필요 없어요. 지우셔도 됩니다.

1구글 시트 만들기

구글 드라이브에서 새 스프레드시트를 하나 만들어요. 이름은 아무거나 (예: "학습결과").

2Apps Script 열고 코드 붙여넣기

시트 상단 메뉴에서 확장 프로그램 → Apps Script 클릭.

화면에 있던 코드를 전부 지우고, 아래 코드를 복사해 붙여넣은 뒤 저장(💾).

🔁 이미 배포하신 분은 이 코드로 다시 붙여넣고 저장한 뒤, 배포 → 배포 관리 → ✏ → 버전 "새 버전" → 배포 를 해야 이름 칸이 적용돼요.

3배포하기 (웹 앱)

오른쪽 위 배포 → 새 배포 클릭.

  • 톱니바퀴 ⚙ → 웹 앱 선택
  • 실행 계정: 나
  • 액세스 권한: 모든 사용자 ← 이거 꼭!
  • 아래 배포 버튼

⚠중간에 "확인되지 않은 앱" 화면이 떠요

겁내지 마세요. 내가 만든 거라 안전해요. 이렇게 통과합니다:

  • 액세스 권한 검토 → 내 구글 계정 선택
  • "Google에서 확인하지 않은 앱" 화면 → 왼쪽 아래 고급 클릭
  • (안전하지 않음)으로 이동 클릭 → 허용

4내 주소(URL) 복사

배포가 끝나면 웹 앱 URL이 나와요. .../exec 로 끝나는 그 주소를 복사해요. 이게 내 시트로 가는 열쇠예요.

5잘 되는지 확인 (여기서 폰으로도 OK)

아래 연결 테스트를 열고, 방금 복사한 주소를 붙여넣고 버튼을 눌러요. 내 시트에 연결테스트 줄이 생기면 성공!

🔌 연결 테스트 열기

6학생용 링크 만들어 나눠주기

링크 생성기에 내 주소를 붙여넣으면 도구별 학생용 링크가 만들어져요. 그 링크를 학급 단톡/클래스룸에 공유하면 끝.

🔗 링크 생성기 열기

학생은 링크로 도구를 풀고 → 끝나면 반 + 번호를 넣고 제출해요. 결과는 내 시트로 들어와요(폰 구글 스프레드시트 앱으로 확인).

❓ 안 될 때 (자주 있는 2가지)
  • 시트에 아무것도 안 들어와요 → 3단계에서 액세스 권한이 "모든 사용자"가 맞는지 확인. 아니면 다시 배포.
  • 코드를 고친 뒤부터 안 돼요 → 배포 → 배포 관리 → ✏ → 버전 "새 버전" → 배포 를 해야 반영돼요.
