# result-collector

학습도구(정적 HTML, GitHub Pages)의 **학생 결과를 각 교사가 자기 구글 시트로 모으는** 범용 위젯입니다.
교사는 코드를 건드리지 않고, **자기 시트 주소가 담긴 링크**를 학생에게 공유하기만 하면 됩니다.

```
[학습도구 링크 ?rc=교사시트주소]  --제출-->  [그 교사의 Apps Script]  -->  [그 교사의 구글 시트]
   반 선택 + 번호 입력                                                    도구별 탭에 한 줄씩
```

- 교사마다 **자기 시트**라 데이터가 섞이지 않고 사생활·할당량 문제 없음
- 도구는 **URL만 바꿔 끼움** → 도구 코드는 한 번만 손봄(전 교사 공용)

---

## A. 담당자(제작자) — 한 번만

1. **이 폴더를 `result-collector` GitHub 저장소로 올리고 Pages 켜기**
   → `https://hongyul67-cpu.github.io/result-collector/collector.js` 와 `.../link-builder.html` 공개
2. **각 도구에 스크립트 2줄 넣기** (도구마다 1회):
   ```html
   <!-- </body> 앞 -->
   <script src="https://hongyul67-cpu.github.io/result-collector/collector.js"
           data-tool="자동화설비 필기"></script>
   ```
   그리고 채점이 끝나는 코드 위치에서:
   ```js
   ResultCollector.open({ score: 85, correct: 17, total: 20, wrong: [3,7,12], durationSec: 242 });
   ```
   > endpoint는 여기 **안 적음**. 교사별 링크의 `?rc=`로 주입됩니다.
3. **`link-builder.html`의 `TOOLS` 배열**에 결과수집을 붙인 도구를 한 줄씩 등록
4. **`Code.gs` 파일을 교사들에게 배포**(카톡/메일). 교사가 자기 시트에 붙여넣을 코드입니다.

---

## B. 일반 교사 — 각자 자기 시트 연결 (배포용 가이드)

### 1단계. 내 시트 만들기 (처음 한 번, 약 5분)
1. 구글 드라이브에서 **새 스프레드시트** 만들기 (이름: 예 "학습결과")
2. 메뉴 **[확장 프로그램] → [Apps Script]**
3. 기본 코드 지우고, 담당자가 준 **`Code.gs`** 붙여넣고 저장(💾)
4. **[배포] → [새 배포]** → 톱니바퀴 ⚙ → 유형 **웹 앱**
   - 실행 계정: **나**
   - 액세스 권한: **모든 사용자**
5. **[배포]** 누르고 권한 승인 → 나오는 **웹앱 URL**(`.../exec`) **복사**

### 2단계. 내 링크 만들기
1. `링크 생성기`(`link-builder.html`) 열기
2. 1단계에서 복사한 **주소 붙여넣기**, 반 목록·번호 최대 확인
3. 도구별로 생긴 **공유 링크 [복사]**

### 3단계. 학생에게 링크 공유 → 결과 확인
- 학생은 그 링크로 도구를 풀고, 끝나면 **반+번호** 넣고 제출
- 결과는 **내 구글 시트**에 도구별 탭으로 자동 저장 (또는 아래 조회 페이지로 확인)

> 코드 수정 후 재배포: [배포]→[배포 관리]→연필 ✏ → 버전 "새 버전" → 배포

## 파일 구성

| 파일 | 용도 | 누가 |
|---|---|---|
| `Code.gs` | 구글 시트 백엔드(도구별 탭 자동 생성) | 교사가 자기 시트에 |
| `collector.js` | 도구에 붙는 제출 위젯. `?rc=`로 교사 시트 주입 | 담당자가 1회 배포 |
| `link-builder.html` | 교사가 자기 링크 뽑는 페이지 | 교사가 사용 |
| `demo.html` | 위젯 동작 확인용 데모 | 테스트 |
| `viewer.html` | (예정) 교사 코드로 자기 결과만 조회 | 교사가 사용 |

## URL 파라미터 (링크 생성기가 자동 생성)

| 파라미터 | 뜻 | 예 |
|---|---|---|
| `rc` | 교사 Apps Script exec URL | `?rc=https://script.google.com/.../exec` |
| `cls` | 반 목록 | `&cls=1,2,3` |
| `max` | 번호 최대 | `&max=40` |
