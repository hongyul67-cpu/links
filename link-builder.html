결과수집 링크 생성기

본인 구글 시트에 학생 결과를 모으려면, 아래에 본인 Apps Script 주소만 붙여넣으세요. 도구별 공유 링크가 자동으로 만들어집니다.

❓ 주소가 없거나 설정이 처음이신가요?
구글 시트 연결부터 차근차근 안내해요. 설정 가이드 →
처음이신가요? — 내 시트 만들고 주소 얻는 법 (5분)
  1. 구글 드라이브에서 새 스프레드시트 만들기 (이름: 예 "학습결과")
  2. 메뉴 [확장 프로그램] → [Apps Script] 클릭
  3. 기본 코드 지우고, 담당자가 준 Code.gs 내용 붙여넣고 저장
  4. [배포] → [새 배포] → 유형: 웹 앱
  5. 실행 계정 나, 액세스 권한 모든 사용자 → 배포
  6. 나오는 웹앱 URL(.../exec)을 복사해 위 칸에 붙여넣기

💡 링크가 긴 건 정상이에요(안에 시트 주소가 들어있어요). 학생에겐 QR을 보여주면 스캔만 하면 돼요.
